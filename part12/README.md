# Full Stack Open — Containers (Part 12)

This section of the course focuses on **using containers to run and manage applications consistently across different environments**. It introduces the fundamentals of Docker and Docker Compose, including:

* **Images vs. Containers** – understanding the difference and lifecycle.
* **Core Docker Commands** – building, running, stopping, and inspecting containers.
* **Volumes & Bind Mounts** – managing persistent data and syncing with local development environments.

> Using containers ensures that applications run the same way on your local machine and on servers.
