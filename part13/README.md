# 📘 Fullstack Open – Part 13: Relational Databases

## Overview

This project is part of the **Fullstack Open** course, Part 13 – *Relational Databases with Sequelize and PostgreSQL*.
It is a **backend application** built with Node.js, Express, Sequelize, and PostgreSQL, focusing on relational database concepts.

Unlike previous exercises, this is a **standalone repository** for practicing database integration, authentication, and testing.

Repository: [https://github.com/mykola-lp/fs-psql](https://github.com/mykola-lp/fs-psql)


* **Relational databases with Node.js** – how to use PostgreSQL instead of MongoDB.
* **Sequelize ORM** – defining models, creating associations, and querying data.
* **Database schema management** – using migrations and controlled updates instead of automatic sync
